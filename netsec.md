```
netstat      -> cek ip routing
ip route     -> cek ip routing
ifconfig     -> cek local ip & MAC address
nmap         -> cek port
iptables     -> cek firewall
dig          -> resolve dns
ip neighbour -> cek ARP
arp          -> cek ARP
```
